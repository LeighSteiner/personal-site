SlyDv

SlyDv is an accessible, secure presentation-material tool aimed specifically at developers. It creates markdown-friendly slide shows complete with an in-slide repl to run code without leaving the presentation, a chat section to take and respond to audience questions, and the ability to share slides post presentation or keep them private. You can here about it here or check out the code here.

Klen-Secure

With over two thousand downloads on npmjs.com, Klen-Secure is a flexible module for handling back end authorizations. Klen-Secure won the “Nerd Cred: Most Technically Challenging” award at the 1706-Stackathon. Hear me talking about why it’s so rad here

Klen-Secure 2

I wanted Klen-Secure to be truly flexible, which means freeing it from as many library dependencies as possible. Klen Secure 2 features a combined checkAuths and whichAuth function, more robust code, and a configuration setting for use with Koa.js.
